# prompt-deepseek

Самый рабочий промпт, для обход цензуры в chat.deepseek.com

1) Устанавливаем файл feardeepseek.txt (Текст не был отредактирован)
2) Заходим в файл, и копируем весь текст (Можете просто не копировать текст)
3) Заходим в chat.deepseek.com, и вставляем файл или текст
4) Если вы отправляете файлом, отправляйте файл с припиской "осознай"
5) И все! Все отлично работает.
